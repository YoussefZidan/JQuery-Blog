Form Requirements:

  • Email field:
  • Required
  • Valid email format using RegExp
  • Password field:
    • Required
    • Min. 6 chars
    • Max 12 chars
    • Should have Letters upper and lowercase, number & special character at the same time

Blog Managment Page Requirements:

  • First column: post name
  • Second column: post content
  • Third column has the following actions:
    • Edit
    • Delete

On click edit post navigate to another page has a form with the fields of:

  • Post name field
  • Post content field
  • If the user make any of the fields empty it should not be submitted or saved
  • If the user click on save should send a request with edit post
